How to use complex numbers

  • Use 'z' to reference the variable number
  • z.modulus() - Returns distance of number from origin
  • z.argument() - Returns angle (-π – π) between number and positive x axis
  • z.conjugate() - Returns complex conjugate of number
  • num(Re, Im) - Returns a complex number with real part Re and imaginary part Im
  • cAdd(z, w) - Returns the result of adding numbers z and w
  • cMult(z,w) - Returns the result of multiplying numbers z and w
